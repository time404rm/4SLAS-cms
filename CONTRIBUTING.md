# Участие в разработке

Спасибо, что хотите помочь проекту!

## Как предложить изменение

1. Сделайте форк репозитория
2. Создайте ветку для своей feature: `git checkout -b feature/your-feature`
3. Внесите изменения
4. Запушьте: `git push origin feature/your-feature`
5. Откройте Pull Request

## Code Style

- Проект на чистом PHP 8 — без фреймворков
- PSR-1 и PSR-4 для автозагрузки
- Имена файлов в `snake_case.php`
- Все пользовательские строки через функцию `__('...')`
- SQL-запросы только через PDO с prepared statements
- Весь вывод через `htmlspecialchars()` / `h()`

## Что важно

- Не добавляйте CDN — все библиотеки локальные
- Не добавляйте внешние HTTP-запросы (кроме настраиваемой Metriки)
- Сохраняйте обратную совместимость конфигурации
