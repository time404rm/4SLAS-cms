# 4SLASeditor — свой WYSIWYG-редактор, которого не стыдно

*Июнь 2026*

Когда речь заходит о редакторе для CMS, все сразу думают про TinyMCE, CKEditor или Quill. Это понятно — они мощные, проверенные, с кучей плагинов. Но у них есть один нюанс: они чужие. Тяжёлые. С зависимостями. И когда что-то идёт не так — ты лезешь в документацию на английском, а не в свой собственный код.

Поэтому для **4SLAS CMS** мы написали свой редактор.

Встречайте **4SLASeditor** (он же 4Tim-редактор) — WYSIWYG-редактор на `contenteditable`, 1383 строки чистого JavaScript, без единой внешней зависимости. Работает в любом современном браузере, весит копейки и делает ровно то, что нужно для ведения блога.

---

## Как это работает

Редактор оборачивает обычный `<div contenteditable="true">` и добавляет к нему тулбар, статус-бар и кучу обработчиков событий. При отправке формы содержимое синхронизируется в скрытый `<textarea>`, который и уходит на сервер.

Инициализация — одна строка:

```javascript
new SimpleEditor('editor', 'content');
```

Где `editor` — id редактируемого div, а `content` — id скрытого textarea.

Всё. Больше ничего не нужно. Стили лежат в `admin.css`, API для загрузки картинок — в `api/upload_image.php`.

## Тулбар — 30 кнопок, иконки SVG

Тулбар строится динамически при создании редактора. Никакого HTML в разметке страницы — всё генерируется в JavaScript.

Все иконки — **встроенные SVG**. Ни одной эмодзи, ни одной картинки. Иконки масштабируются вместе с интерфейсом, не пикселят, меняют цвет вместе с темой.

Кнопки сгруппированы разделителями:

### Форматирование текста
**B** / *I* / <u>U</u> / <s>S</s> — жирный, курсив, подчёркнутый, зачёркнутый. Работают через `document.execCommand`, поддерживают хоткеи `Ctrl+B`, `Ctrl+I`, `Ctrl+U`.

### Списки
Маркированный и нумерованный. Отдельные кнопки для повышения/понижения отступа (Tab/Shift+Tab).

### Выравнивание
Влево, по центру, вправо, по ширине. Четыре кнопки с иконками-линиями.

### Цвет
Цвет текста и цвет фона. При клике открывается нативный `<input type="color">` — никаких кастомных пикеров, только системный диалог.

### Блоки
- **H2, H3** — заголовки
- **Цитата** — `<blockquote>`
- **Форматированный текст** — `<pre>` (моноширинный, для кода и логов)
- **Горизонтальная линия** — `<hr>`

### Вставка
- **Ссылка (Ctrl+K)** — модальное окно: URL, текст, target
- **Якорь** — внутристраничная закладка `<a class="editor-anchor">`
- **Загрузка изображения** — drag-and-drop или выбор файла → AJAX-загрузка → вставка. Поддерживается копипаст картинок из буфера
- **URL изображения** — вставить картинку по ссылке
- **Загрузка файла** — PDF, DOC, ZIP. Файл загружается на сервер, в контент вставляется ссылка
- **Видео** — оверлей с предпросмотром YouTube/VK/Rutube
- **Таблица** — вставка с выбором размеров строк/столбцов

### Код и символы
- **Вставка кода** — `<code>` с подсветкой
- **Смайлы** — системный emoji-picker ОС
- **HTML-режим** — переключение между визуальным и исходным кодом с нумерацией строк

### Действия
- **Отмена (Ctrl+Z)** / **Повтор (Ctrl+Y)**
- **Полноэкранный режим**

Справа на тулбаре — версия редактора: *«4tim редактор v0.3»*.

## Умная вставка — никакого мусора из Word

Одна из главных болей WYSIWYG-редакторов — когда копируешь текст из Word или с сайта, и вместе с текстом прилетает куча `<span style="font-size: 14pt; font-family: 'Times New Roman'; color: rgb(51, 51, 51); ...">`.

4SLASeditor чистит вставляемый HTML:

- **Удаляются** теги `<style>`, `<link>`, `<meta>`, `<title>`
- **Удаляются** Word-классы (`class="Mso..."`)
- **Удаляются** inline-стили, кроме: `color`, `background-color`, `font-weight`, `font-style`, `text-decoration`, `text-align`
- **Конвертируются** теги `<font>` в `<span>`
- **Конвертируются** теги `<b>` и `<i>` в `<strong>` и `<em>`
- **Удаляются** запрещённые теги: `script`, `iframe`, `object`, `embed`
- **Вырезаются** `on*`-атрибуты и `javascript:` URL

Результат: чистый HTML, который не сломает вёрстку.

Если HTML не удалось распарсить — вставляется plain text. Всё безопасно.

## Режим исходного кода

Нажал кнопку `HTML` — и редактор превращается в пару `<textarea>` + нумерация строк слева.

- Tab вставляет 4 пробела (не переключает фокус)
- Шрифт моноширинный, фон чуть темнее
- При переключении обратно в визуальный режим содержимое парсится и отображается как HTML

Нумерация строк синхронизируется со скроллом — номера всегда на тех же строках, что и код. Всё на CSS + `scrollTop`.

## Таблицы — с контекстным меню

При вставке таблицы появляется всплывающее меню управления:

- Добавить строку сверху / снизу
- Добавить столбец слева / справа
- Удалить строку / столбец
- Удалить таблицу

Меню появляется рядом с активной ячейкой. Таблица рисуется с `border: 1px solid #ccc` и базовыми стилями для ячеек.

## Изображения: загрузка, drag-and-drop, редактирование

Три способа вставить картинку:

1. **Кнопка «Загрузить»** — открывает `<input type="file">`, картинка уходит через AJAX на `api/upload_image.php`, сервер возвращает URL, редактор вставляет `<img>`
2. **Перетаскивание** — бросил файл в редактор → AJAX → вставка
3. **Вставка из буфера** — Ctrl+V скриншота или скопированной картинки → blob → AJAX

При клике на картинку она выделяется рамкой. Повторный клик — модальное окно редактирования:

- **URL** — поменять источник
- **Ширина** / **Высота** — в пикселях или процентах
- **Alt-текст** — для доступности и SEO
- **Удалить** — убрать картинку

Все картинки при вставке оборачиваются в `<div class="image-container">`, чтобы с ними работал Lightbox (при просмотре на фронтенде).

## Ссылки и якоря

**Ссылка** — модальное окно:
- URL (обязательно)
- Текст (если выделен — подставляется)
- Открывать в новом окне (чекбокс)

Двойной клик по существующей ссылке — окно редактирования.

**Якорь** — внутристраничная метка. Вставляется как `<a class="editor-anchor" id="...">`. Невидим в браузере, но работает для навигации по странице.

---

## Видео и файлы

**Видео** — вставляется через модальное окно. Поддерживаются:
- YouTube (`youtube.com/watch?v=...` или `youtu.be/...`)
- VK Video (`vk.com/video...`)
- Rutube (`rutube.ru/video/...`)

Вставляется как `<div class="video-container">` с iframe внутри. На фронтенде видео адаптивное (16:9).

**Файлы** — загружаются через `api/upload_file.php`. Максимальный размер — 20 МБ. В контент вставляется ссылка на скачивание с иконкой типа файла и размером. Типы файлов: PDF, DOC, DOCX, XLS, XLSX, ZIP, RAR, TXT.

## Полноэкранный режим

Кнопка «На весь экран» — редактор разворачивается на весь viewport.

- Тулбар фиксируется сверху
- Скролл только внутри редактора
- Кнопка меняет иконку (свёрнуть/развернуть)
- Escape — выход из полноэкранного режима

Удобно для длинных статей.

## Синхронизация с формой

Редактор сам находит родительскую `<form>` и вешает на неё обработчик `submit`. При отправке формы содержимое редактора копируется в скрытый `<textarea>`.

Вручную можно вызвать `editor.syncToHidden()` — например, перед AJAX-сохранением.

## Безопасность

Редактор работает с HTML, поэтому безопасность критична:

- **Санитайз при вставке:** удаление скриптов, запрещённых тегов, вредоносных атрибутов
- **Санитайз при источнике:** при переключении из HTML-режима в визуальный контент тоже чистится
- **Серверный санитайз:** в `post_edit.php` контент проходит через `strip_tags` с разрешёнными тегами (уровень CMS, не редактора)
- **Загрузка изображений:** MIME-проверка через `finfo`, белый список расширений

## Почему свой, а не готовый

| Критерий | TinyMCE | 4SLASeditor |
|---|---|---|
| Размер (минифицированный) | ~600 КБ | ~45 КБ |
| Зависимости | moment.js, jQuery (в старых) | **0** |
| Контроль над кодом | Чужой репозиторий | **Твой файл** |
| Кастомизация | Плагины + конфиг | **Правишь код** |
| Русский интерфейс | Есть, но не везде | **Весь на русском** |
| Вставка из Word | Чистит | **Чистит** |
| Картинки drag-and-drop | Есть | **Есть** |
| Полноэкранный режим | Есть | **Есть** |
| Таблицы | Расширенные | **Базовые + контекстное меню** |
| Бесплатно | Community-версия | **MIT** |

Главное преимущество — ты контролируешь каждую строчку. Нужна фича — дописал 20 строк. Не нравится поведение — поправил. Не надо ждать обновлений, не надо разбираться в чужой архитектуре на 50 000 строк.

## Как попробовать

Редактор встроен в 4SLAS CMS — достаточно установить систему и зайти в админку. Но можно использовать и отдельно:

```html
<div id="editor" contenteditable="true"></div>
<textarea id="content" name="content" style="display:none"></textarea>
<script src="4SLASeditor.js"></script>
<script>
  new SimpleEditor('editor', 'content', {
    apiBase: '/api/'
  });
</script>
```

Подключи `admin.css` (или напиши свои стили для `.editor-toolbar`, `.custom-editor`, `.editor-source-wrap`) — и редактор готов к работе.
